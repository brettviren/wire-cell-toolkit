# BlobDeclustering

Extracts all blob nodes from an input cluster graph and returns them as a flat IBlobSet, effectively reversing the clustering step.

## Node Properties

| Property | Value |
| --- | --- |
| Factory name | `BlobDeclustering` |
| Concrete class | `WireCell::Img::BlobDeclustering` |
| Node category | function |
| Primary interface | `IBlobDeclustering` |
| Input type(s) | `ICluster` |
| Output type(s) | `IBlobSet` |
| Configurable | no |

# WireGenerator

Generates all wires for a three-plane (U, V, W) detector geometry from an IWireParameters input by computing intersections of wire directions with the detector bounding box.

## Node Properties

| Property | Value |
| --- | --- |
| Factory name | `WireGenerator` |
| Concrete class | `WireCell::WireGenerator` |
| Node category | function |
| Primary interface | `IWireGenerator` |
| Input type(s) | `IWireParameters` |
| Output type(s) | `IWire::vector` |
| Configurable | no |

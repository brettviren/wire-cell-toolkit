# WireSummarizer

Builds an IWireSummary lookup structure from an input collection of wires, enabling fast per-channel and per-plane wire queries downstream.

## Node Properties

| Property | Value |
| --- | --- |
| Factory name | `WireSummarizer` |
| Concrete class | `WireCell::WireSummarizer` |
| Node category | function |
| Primary interface | `IWireSummarizer` |
| Input type(s) | `IWire::vector` |
| Output type(s) | `IWireSummary` |
| Configurable | no |
